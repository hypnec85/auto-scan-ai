1. Streamlit 앱 종료 명령어:

   Get-Process python | Where-Object { $_.Path -like "*streamlit*" } | Stop-Process
  설명: 이 명령어는 현재 실행 중인 모든 python 프로세스 중에서 경로에 streamlit이 포함된 프로세스를 찾아 강제로 종료합니다. 이를 통해 Streamlit 앱 서버가 완전히 중지됩니다.

  2. Streamlit 앱 실행 명령어:

   Start-Process python -ArgumentList "-m streamlit run app.py --server.headless true" -NoNewWindow
  설명: 이 명령어는 streamlit run app.py 명령을 PowerShell의 Start-Process cmdlet을 사용하여 백그라운드에서 실행합니다. --server.headless true 옵션은 Streamlit이 시작될 때 웹 브라우저를 자동으로 열지 않고, 콘솔에서     
  불필요한 질의(예: 이메일 입력)를 건너뛰도록 합니다. -NoNewWindow는 별도의 콘솔 창을 띄우지 않고 현재 세션의 백그라운드에서 실행되도록 합니다.
